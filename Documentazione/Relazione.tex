    \section{Nozioni di logica}
    La logica permette di avere un linguaggio (anche formale) utilizzato per descrivere i vari elementi.

    Il \textbf{linguaggio del primo ordine} è un linguaggio che è formato da un certo insieme di simboli:
    \begin{itemize}
      \item \textit{relazionali}: $p, q, r, \dots$
      \item \textit{funzionali}: $f, g, h, \dots$
      \item \textit{costanti}: $c, d, \dots$
      \item \textit{logici}: insieme di simboli
      \begin{itemize}
        \item parentesi tonde ( ) e virgola ,
        \item insieme numerabile ($\mathbb{N}$) di variabili ($v, x, y, \dots$) 
        \item insieme di connettivi logici ($\land, \lor, \Rightarrow, \Leftrightarrow$)
        \item quantificatori ($\forall, \exists$)
      \end{itemize}
    \end{itemize}
